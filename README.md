##Contributing

  1. Fork the repo via Github
  2. `git clone` your fork
  3. Run `rake setup`
  4. Make some changes
  5. Tests the changes `bundle exec rake`
  6. Submit a Pull Request on github (We would like to review the code)
  7. Keep being AWESOME!
